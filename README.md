
# Segunda Pele Tattoo — Site

Stack: Next.js (App Router) + Tailwind.

## Rodar local
```bash
npm install
npm run dev
# abre http://localhost:3000
```

## Deploy na Vercel
- Suba essa pasta para um repositório no GitHub.
- Na Vercel: Add New Project → importe o repo → Framework: Next.js → Deploy.
